What is SolversLib?

SolversLib is an updated and maintained fork of the older FTCLib. PRs made to both SolversLib & FTCLib are regularly merged into SolversLib.

Learn more
Native Javadocs

SolversLib features automatically updating Javadocs for each module.

- SolversLib Core (standard FTCLib, has Command Base, Hardware, Gamepads, etc.)

- SolversLib Pedro Pathing (Pedro Pathing support in FTCLib)

Insight

The SolversLib

SolversLib is the FTC library to end all libraries. By porting features from WPILib for FRC, SolversLib makes the use of advanced systems extremely easy for even rookie programmers.
